# ac-ten-bo

Tenant Back Office for staff operations.

## Overview

Staff back office dashboard for:
- Service request management
- Technician scheduling
- Customer lookup
- Inventory tracking

## Port

- **3002** (Development)

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Setup

```bash
pnpm install
cp .env.example .env
pnpm dev
```
